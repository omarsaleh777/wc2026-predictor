# 🏆 World Cup AI Predictor (AI-Orchestrated ML Pipeline)

A robust, Pure Logic machine learning pipeline designed to predict World Cup match outcomes and scorelines. This project demonstrates advanced **AI Orchestration**, where the actual codebase was generated by Large Language Models (Claude & Gemini), while I served as the **System Architect and Lead Data Engineer**.

## 🧠 Project Philosophy & Transparency
In modern software engineering, coding syntax is a commodity. The true challenge is system architecture and constraint management. 

This repository was built using **Prompt Engineering and AI Orchestration**. My direct contributions were:
*   **System Architecture:** Designing the pipeline flow (ETL -> Train -> Predict -> Streamlit UI).
*   **Feature Engineering Strategy:** Designing the mathematical logic for features like "Strength of Schedule" and "The Defensive Wall" to reflect real football tactics.
*   **Architectural Defense:** Strictly preventing the AI from introducing artificial class weights (e.g., forcing draw predictions) to maintain the integrity of raw statistical probabilities.
*   **Debugging & Data Leakage Prevention:** Identifying wide-format data constraints and enforcing chronological `shift(1)` operations to prevent look-ahead bias.

## ⚙️ Core Architecture & Features

### 1. The Logical Bridge (Custom XGBoost Wrapper)
To prevent XGBoost's strict label constraints from breaking the Streamlit UI rounding logic, a custom `CustomXGBClassifier` wrapper was architected. It trains on standard `[0, 1, 2]` constraints but outputs mapped `[-1, 0, 1]` predictions to maintain pure system logic.

### 2. Feature Engineering Highlights
*   **Exponential Time Decay:** Applied a time-decay weight formula ($k=0.00019$) to historical matches, forcing the model to respect modern football tactics over ancient history.
*   **The Defensive Wall:** Engineered rolling averages for `goals_conceded_per_match` and `clean_sheet_ratio` handling wide-format data matrices perfectly.
*   **Match Tension Index:** Calculated `elo_gap` and `defensive_parity` to allow the model to organically sense tightly matched teams without artificial weights.

### 3. Dual-Branch Prediction
*   **Outcome Classifier:** Predicts Home Win / Draw / Away Win based on pure probability distributions.
*   **Scoreline Regressors:** Two distinct Continuous Regressors (Home Goals & Away Goals) achieving an MAE of < 1.0 per match.

## 🛠️ Tech Stack
*   **Machine Learning:** XGBoost, Scikit-learn, Pandas, NumPy
*   **UI/Frontend:** Streamlit
*   **Paradigm:** AI Orchestration & Prompt Engineering

# 🏆 متنبئ مباريات كأس العالم بالذكاء الاصطناعي 

نظام تعلم آلة متقدم مصمم لتوقع نتائج مباريات كأس العالم والأهداف بدقة عالية بناءً على "المنطق الرياضي البحت". يعكس هذا المشروع مهارات **إدارة وتوجيه الذكاء الاصطناعي (AI Orchestration)**، حيث تم توليد الأكواد البرمجية بواسطة وكلاء الذكاء الاصطناعي (Claude & Gemini)، بينما تركز دوري في **الهندسة المعمارية للنظام (System Architecture)** وإدارة منطق البيانات.

## 🧠 فلسفة المشروع والشفافية
في هندسة البرمجيات الحديثة، كتابة الكود أصبحت أداة، بينما التفكير المعماري هو الأساس. تم بناء هذا المشروع عبر توجيه الذكاء الاصطناعي. تركزت جهودي الهندسية في:
*   **هندسة النظام:** تصميم تدفق البيانات بين معالجة البيانات (ETL)، تدريب النماذج، وواجهة المستخدم (UI).
*   **هندسة الميزات (Feature Engineering):** ابتكار وبناء المنطق الرياضي لخصائص مثل "الكفاءة الهجومية" و"الجدار الدفاعي".
*   **حماية المعمارية:** الرفض الصارم لمحاولات الذكاء الاصطناعي استخدام الأوزان المصطنعة لتحسين توقعات "التعادل"، وذلك لحماية دقة الاحتمالات الإحصائية الخام.
*   **منع تسريب البيانات:** تطبيق تقنيات المعالجة الزمنية `shift(1)` لضمان عدم اطلاع النموذج على أهداف المباراة قبل توقعها.

## ⚙️ المعمارية الهندسية

*   **الجسر المنطقي (Custom Wrapper):** تمت كتابة غلاف مخصص لخوارزمية `XGBoost` للتحايل على قيود تصنيف البيانات، مما يضمن خروج النتائج بصيغة تدعم خوارزمية العرض في واجهة Streamlit.
*   **معامل الاضمحلال الزمني:** تطبيق معادلة رياضية تعطي وزناً أعلى للمباريات الحديثة، مما يجبر النموذج على استيعاب التكتيكات الكروية المعاصرة.
*   **توقع دقيق للأهداف (Regressors):** استخدام نماذج انحدار منفصلة لتوقع أهداف المضيف والضيف، مما حقق نسبة خطأ (MAE) تقل عن هدف واحد للمباراة.
